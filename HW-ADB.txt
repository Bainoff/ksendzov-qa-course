1) Отобразить подключенный девайс в консоли.
.\adb devices
2) Вывести адрес приложения todolist в системе Android
.\adb shell 'pm list packages -f' | findstr todolist
3)  Установить .apk файл приложения todolist на телефон с компьютера через  ADB
.\adb install d:\android\apps\apk
4) Сделать скриншот запущенного приложения todolist и сразу скопировать на компьютер в одной команде.
.\adb shell screencap /storage/emulated/0/DCIM/scr.png | .\adb pull /storage/emulated/0/DCIM/scr2.png D:/courses/ksendzov/android/scr2.png
5) Вывести в консоль логи приложения todolist
.\adb shell logcat -d | findstr todolist
6) Скопировать логи приложения todolist на компьютер
.\adb shell logcat -d | findstr todolist > func.logcat
7) Удалить приложение todolist с телефона через ADB
.\adb uninstall com.developer.todolist
